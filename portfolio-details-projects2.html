Details

The Coin Quest: Raindow Rush Game is an engaging online game (inside a bank app) where players can collect coins, pots and rainbows. The objective is to collect as many coins as possible, which get transferred from the main account to the savings account after the game. The idea is to motivate bank-customers to save more money. This project is a fun way to introduce basic banking concepts in a gamified environment. The game is built using Python with Flask for the backend, including Sqlite3 for the database, and for the frontend i used HTML, CSS, and JavaScript(and Phaser.js), providing a smooth and interactive user experience.
