Machine Learning for Smarter Wastewater Management

I work as project manager on Bergen kommune's large-scale initiative to optimize the wastewater infrastructure using machine learning and cloud technology.

The solution collects real-time data from millions of sensors through Azure Data Factory, primarily monitoring wastewater levels across the city’s network of over 260 pumping stations.

ETL processes, feature engineering, and ML training are performed in Azure Databricks. The outputs are visualized in dynamic dashboards built in Microsoft Power BI, which provide engineers, planners and operations staff with deep insight into flow behavior, overflow risk, and long-term trends.

Although the project was initially developed by a skilled team, I have taken full responsibility for the pipeline — from data ingestion and orchestration to notebook logic and dashboard delivery — and now ensure the system runs reliably and evolves with new needs.

The system enhances decision-making, improves maintenance routines, and contributes to more sustainable water and wastewater management in Bergen.
